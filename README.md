[Nest](https://github.com/nestjs/nest) Proyecto Inicial en NestJs

## Project setup

```bash
$ npm install
```

```bash
$ npm run setup
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
